# BharatOS Hardware Support

## Supported Architectures

### x86-64 (Primary)
- Intel Core i3/i5/i7/i9 (6th gen+)
- Intel Xeon (Scalable, Cascade Lake+)
- AMD Ryzen (1000 series+)
- AMD EPYC (Rome, Milan, Genoa)
- Legacy: Intel Core 2 Duo, AMD Phenom

### ARM64 (Secondary)
- Apple Silicon (M1, M2, M3, M4)
- Qualcomm Snapdragon 8cx Gen 2/3
- Apple A12+
- Raspberry Pi 4/5 (aarch64)
- AWS Graviton, Ampere

### RISC-V (Experimental)
- SiFive U74, S76
- Alibaba XuanTie
- StarFive JH7110

## Graphics

### Intel
- Integrated: UHD Graphics 600+, HD Graphics 400+
- Arc series (Alchemist, Battlemage)

### AMD
- RDNA 2: RX 6000 series
- RDNA 3: RX 7000 series
- Radeon Pro series
- APUs: Ryzen 4000/6000/8000 G-series

### NVIDIA
- GeForce RTX 20/30/40/50 series
- Quadro / RTX professional
- Tegra (Jetson, Shield)

## Storage

### SSD
- NVMe (PCIe 4.0, 5.0)
- SATA III (6 Gbps)
- U.2
- M.2 (B, M, B+M keys)

### HDD
- SATA III
- SAS
- USB external

## Networking

### Ethernet
- 1/2.5/5/10/25/40/100 GbE
- Intel i210/i226/i219, I225-V
- Realtek RTL8111/8125
- Broadcom NetXtreme

### Wi-Fi
- 802.11ax (Wi-Fi 6/6E/7)
- Intel AX200/AX210/AX411
- MediaTek MT7921/7922
- Qualcomm FastConnect

### Bluetooth
- 5.0, 5.1, 5.2, 5.3, 5.4
- Intel AX* Bluetooth
- Qualcomm WCN

## Peripherals

### Input
- USB keyboard/mouse
- Touchpad/touchscreen (multi-touch)
- Pen/stylus (Wacom-compatible)
- Gamepad (XInput, DirectInput)
- Trackpoint (ThinkPad)

### Audio
- HDA codecs: Realtek ALC* series
- USB Audio (UAC 1.0/2.0)
- Bluetooth A2DP/HFP/HSP
- HDMI/DP audio

### Webcam
- UVC-compatible devices
- Hello/IR cameras

### Printers
- IPP (Internet Printing Protocol)
- USB Printer Class
- AirPrint-compatible
